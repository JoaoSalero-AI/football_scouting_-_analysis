Metrics glossary
  • SDR%: Successful Dribble Rate (successful dribbles / attempted dribbles).
  • SDR: Successful dribbles (count).
  • xG: Expected goals.
  • xA: Expected assists.
  • GI: Goal involvement = goals + assists.
  • xGI: Expected goal involvement = xG + xA.
  • TOS: Total shots.
  • SOT: Shots on target.
  • BCM: Big chances missed.
  • KEYP: Key passes (passes leading to a shot).
  • BCC: Big chances created.
  • APS: Pass attempts.
  • APS%: Pass completion rate.
  • ALB: Accurate long balls.
  • LBA%: Long ball accuracy.
  • TACK: Tackles.
  • INT: Interceptions.
  • YC: Yellow cards.
